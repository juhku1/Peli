# 3D Selainpeli - Three.js

Yksinkertainen 3D-selainpeli, jossa kerätään kolikoita ja väistellään esteitä.

## Nopea aloitus

1. Avaa `index.html` selaimessa
2. Käytä WASD tai nuolinäppäimiä liikkumiseen
3. Välilyönti hyppäämiseen
4. Kerää kultaisia kolikoita pisteitä varten
5. Vältä punaisia esteitä!

## Kehitysideoita (pyydä tekoälyä toteuttamaan):

### Helpot lisäykset:
- "Lisää ääniefektit kolikoiden keräämiseen"
- "Tee taustamusiikin tuki"
- "Lisää elämäpisteet törmäysten sijaan"
- "Tee pause-toiminto P-näppäimelle"
- "Lisää partikelitehosteita"

### Keskitason lisäykset:
- "Tee vaihtuvat tasot vaikeustasoineen"
- "Lisää vihollisia jotka liikkuvat pelaajaa kohti"
- "Tee 3D-mallit (GLB/GLTF) tuotteiden lataamiseen"
- "Lisää mini-kartta nurkkaan"
- "Tee parhaat pisteet local storageen"

### Edistyneet lisäykset:
- "Lisää moninpelituki WebSocketeilla"
- "Tee procedurally generated tasoja"
- "Lisää fysiikkamoottorin (Cannon.js tai Ammo.js)"
- "Tee VR-tuki WebXR:llä"
- "Optimoi suorituskyky instanced meshillä"

## Teknologia

- **Three.js** - 3D-grafiikka
- **Vanilla JavaScript** - logiikka
- **CDN** - ei asennuksia tarvita

- "Tee grafiikkavalikko jolla voi säätää laatua"
